# Easy to use CLI Word Document Word Counter
## After cloning the repo don't forget to cd to the directory
```bash
cd docx-wordCounter
```
## Build the rust app
```bash
cargo build
```
## Run the app
```bash
cargo run
```
### Note : always use absolute path like
###  /home/user/Documents/yourfile.docx
### instead of
### ./yourfile.docx
